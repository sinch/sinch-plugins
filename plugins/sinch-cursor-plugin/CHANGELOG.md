# Changelog

All notable changes to the Sinch Cursor Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-17

### Added
- Initial release of Sinch Cursor Plugin
- Support for sending SMS and RCS messages
- Multi-channel messaging with automatic fallback
- Message type support: text, media, location, choice, template
- Webhook management (list, create, update, delete)
- List active senders and phone numbers
- Webhook triggers reference
- MCP server integration via @sinch/mcp
- Setup and configuration commands
- Comprehensive documentation and examples

### Features
- **Messages API**: Send text, media, location, choice, and template messages
- **Webhooks API**: Full CRUD operations for webhooks
- **Senders API**: List active phone numbers
- **MCP Integration**: Seamless integration with Sinch MCP server
- **Channel Support**: SMS and RCS with automatic fallback
- **Batch Messaging**: Support for sending to multiple recipients
